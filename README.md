# Fluid Fall Simulation Using Pygame
This is a basic simulation of a spherical object falling in a liquid from a given height.

## Features 
* Inputs: Radius, mass, liquid density, viscosity, and drop height.
* Simulates falling patterns based on physical parameters.

## Requirements
* Python 3.x
* pygame

## How to use
1. Ensure Python is installed.
2. Install the required library: `pip install pygame`
3. Run the simulation: `python main.py`
